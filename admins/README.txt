Для добавления списка админов для каждого сервера необходимо создать файл с именем вида: "<IP>;<PORT>.txt".
То есть, для сервера с адресом "5.254.105.122:7777" нужно создать файл "5.254.105.122;7777.txt";
Также можно добавить общий список админов, для этого создайте файл "all.txt".
Если для сервера не будет найден личный файл админов, бот загрузит админов из общего списка, если общего списка не будет, то загрузит с сайта.
Админов в файлы нужно записывать по одному в каждой строке.
